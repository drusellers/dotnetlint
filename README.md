# dotnetlint
A journey into ASTs


## Usage

```
dotnetlint.exe [options] [file]*
```

Such as

```
dotnetlint.exe file1.cs file2.cs
```

or

```
dotnetlint.exe *.cs
```


### With Github

__EXPERIMENTAL__

```
dotnetlint.exe github https://api.github.com/repos/<owner>/<repo>/pulls/<id>/22/files
```
